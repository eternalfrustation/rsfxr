# Rsfxr

A small and simple library for getting procedural sound effects for gamedev.

HEAVILY inspired by the original [jsfxr](https://sfxr.me/).

Currently supported effects:
- Sine wave
- Square wave
- Sawtooth wave
- White Noise
- Frequency cutoff
- Frequency slide
- Vibrato
- Arpeggiation
- Retrigger
- Flanger
- Envelope
